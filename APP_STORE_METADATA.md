## Full Description (iOS & Android)

**Experience the Great War like never before.**

Trenches: WW1 Tactics is a turn-based tactical strategy game that puts you in command of soldiers fighting in the trenches of World War I. Lead your men through 20 historically-inspired missions across the Western and Eastern Fronts.

**FEATURES:**

**Authentic WWI Combat**
- Command infantry, machine gunners, snipers, cavalry, and tanks
- Experience realistic trench warfare with artillery barrages and gas attacks
- Dynamic weather affects battlefield conditions

**Deep Tactical Gameplay**
- Manage supply lines to keep your troops fighting
- Call in reinforcements at critical moments
- Use terrain and cover to your advantage
- Flank enemies and exploit weaknesses

**Compelling Campaign**
- 20 missions spanning 1914-1918
- Play as British, French, German, or American forces
- Experience iconic battles: Verdun, the Somme, Vimy Ridge, and more

**Personal Stories**
- Read letters home from soldiers at the front
- Follow newspaper headlines based on your victories
- Earn medals and watch your veterans grow

**Extensive Progression**
- 50+ achievements to unlock
- Veteran soldiers that grow stronger with experience
- Multiple difficulty levels

**Respectful Treatment**
- Historically accurate scenarios and equipment
- Memorial to honor the fallen
- Educational content about the Great War

## Review Notes for App Store

This is a turn-based tactical strategy game set in World War I. The game features:

- Cartoon/stylized military units (not realistic violence)
- No graphic content - units simply disappear when defeated
- Historical educational content about WWI
- Memorial screen honoring fallen soldiers
- No multiplayer, no chat, no user-generated content
- No in-app purchases in this version
- No ads
- Fully offline gameplay

Demo account not required - game is fully playable from first launch.

## Build Commands

```bash
# Install EAS CLI
npm install -g eas-cli

# Login to Expo
eas login

# Configure project
eas build:configure

# Build for iOS (App Store)
eas build --platform ios --profile production

# Build for Android (Play Store)
eas build --platform android --profile production

# Submit to App Store
eas submit --platform ios

# Submit to Play Store
eas submit --platform android
```
