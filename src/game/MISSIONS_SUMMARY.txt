Data Structure:
===============
Each mission contains:
- Metadata: id, name, location, date, weather, briefing, objective
- Units array: player and enemy units with type, team, x, y coordinates
- Terrain array: map terrain features with type and coordinates
- Victory/Defeat logs: mission outcome messages
- Optional fields: diaryEntry, specialVictory, turnsToSurvive, specialMechanic
